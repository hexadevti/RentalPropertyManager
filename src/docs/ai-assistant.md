# Assistente IA

## O que é esta tela?

Chat inteligente que responde perguntas sobre os dados do seu portfólio em linguagem natural. O assistente acessa em tempo real os registros de propriedades, contratos, finanças, tarefas e outros módulos para fornecer respostas precisas.

## Como usar

1. Digite sua pergunta na caixa de texto
2. Pressione **Enter** para enviar — ou **Ctrl+Enter** para quebrar linha
3. O assistente analisa os dados e responde em alguns segundos

## O que o assistente pode responder

- Saldo financeiro do mês atual e por categoria
- Propriedades disponíveis e ocupadas
- Contratos ativos, vencendo ou encerrados recentemente
- Tarefas pendentes por propriedade ou prioridade
- Agendamentos próximos
- Documentos cadastrados por tipo
- Vistorias realizadas e pendências identificadas

## Perguntas rápidas

Use os botões de **Perguntas rápidas** no painel lateral para enviar consultas comuns com um clique:
- Saldo e maiores categorias de gasto
- Propriedades com contrato ativo
- Tarefas pendentes
- Contratos vencendo ou encerrados

## Contexto carregado

O painel **Contexto carregado** mostra quais módulos foram consultados na última pergunta e quantos registros foram incluídos. O sistema limita o volume de dados para manter respostas rápidas.

## Persistência da conversa

O histórico de mensagens é mantido enquanto você navega pelo sistema. Você pode alternar para outras telas e voltar para continuar a conversa do ponto onde parou.

## Limpar conversa

Use o botão **Limpar conversa** para reiniciar o chat sem histórico.

## Boas práticas

> Seja específico nas perguntas: "Qual o saldo de março?" funciona melhor que "Como estão as finanças?".

> O assistente responde com base nos dados cadastrados — se um registro não foi criado no sistema, ele não aparecerá na resposta.

> Para ações que alteram dados (criar contrato, registrar pagamento), use os módulos correspondentes — o assistente é consultivo, não executa operações.

## Limitações

- O assistente não acessa dados históricos além dos registros carregados (limite de ~120 registros por tabela)
- Não realiza cálculos retroativos que exijam todos os registros históricos
- Respostas dependem da qualidade e completude dos dados cadastrados

## Disponibilidade

O assistente está disponível apenas para usuários com perfil de **administrador aprovado**. O uso é registrado com informações de tokens consumidos para controle de custo.

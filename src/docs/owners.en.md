# Owners

## What is this screen?

Manages the registration of property owners — individuals or companies who own the properties managed by the system.

## Form fields

### Identification
- **Name** — full name (individual) or company name (legal entity)
- **Email** — primary contact for sending reports
- **Phone** — direct contact

### Documentation
Add multiple documents (CPF, CNPJ, ID, passport, etc.), each with type and number.

### Personal data (individual)
- **Nationality** — Brazilian or foreign
- **Marital status** — single, married, divorced, widowed
- **Profession** — current occupation

### Address
Full address for correspondence and formal documents.

### Notes
Internal notes: transfer agreements, co-ownership percentages, contact preferences, etc.

## Linked properties

Each owner displays the list of properties associated with them. The link is made in the **Property** registration, in the "Linked owners" section.

## CSV import

Use the **Import CSV** button to import multiple owners at once. Available fields: name, email, phone, address, nationality, marital status, profession, document type and number.

## Search

Find owners by name, email, or document using the search bar.

## Tips

> A property can have multiple owners (co-ownership). Each one appears separately in this screen.

> Owner data is used in **contract/booking templates** for automatic party information filling.

> Use the notes field to record ownership percentages in co-ownership scenarios.

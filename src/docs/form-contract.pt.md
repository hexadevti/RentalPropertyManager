# Formulário de Contrato/Reserva

## Hóspede e propriedade

- **Hóspede** — selecione o inquilino cadastrado. Se ainda não existir, cadastre primeiro em Hóspedes ou use o botão **Novo Hóspede/Inquilino** no próprio formulário.
- **Propriedades** — selecione uma ou mais propriedades vinculadas (útil para quem aluga múltiplos quartos).
- **Tipo de locação** — Mensal (longo prazo) ou Temporada (diárias com data de saída definida).

## Datas

- **Data de início** — primeiro dia do contrato/reserva
- **Data de fim** — último dia previsto; obrigatório para temporadas

## Financeiro

- **Valor mensal** — aluguel base
- **Dia de vencimento** — dia do mês em que o pagamento é esperado (ex: dia 5 → vence todo dia 5)
- **Condições especiais de pagamento** — cláusulas extras: carência, multa por atraso, desconto por pontualidade etc.

## Observações

Notas internas sobre o contrato/reserva. Não aparecem no PDF gerado.

## Template de Contrato/Reserva

Selecione um template para habilitar a geração de PDF com variáveis preenchidas automaticamente.

> Crie ou edite templates em **Templates de Contrato/Reserva** antes de usar esta opção.

## Geração de PDF

Após salvar, use o botão **Gerar PDF** no card do contrato:
- Selecione o template e o idioma
- Use **Visualizar** para abrir no navegador
- Use **Baixar PDF** para salvar o arquivo

## Contratos/reservas criados via sync iCal

Quando importados pela sincronização de plataformas, os contratos/reservas são criados com:
- Tipo: **Temporada**
- Valor mensal: **0** (edite após importar)
- Hóspede: criado automaticamente com o nome da plataforma
- Observações: nome do feed da plataforma de origem

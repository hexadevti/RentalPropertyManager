# Service Providers

## What is this screen?

Registers vendors, technicians, freelancers, and companies that provide maintenance, repair, or conservation services for the property portfolio.

## Form fields

- **Name** — full name (freelancer) or company name
- **Service type** — specialty (e.g., Electrical, Plumbing, Cleaning, Painting)
- **Phone** — primary contact
- **Email** — for sending requests and quotes
- **Tax ID** — fiscal document for invoice and service contracts
- **Address** — location for formal correspondence
- **Notes** — quality rating, average pricing, access restrictions, etc.

## CSV import

Use the **Import CSV** button to import multiple service providers at once. Available fields: name, service, phone, email, document, address, and notes.

## Links with other modules

| Module | How to use |
|---|---|
| **Appointments** | Schedule a maintenance visit with the provider |
| **Finances** | Record an expense paid to the provider |
| **Contracts/Bookings** | Reference the provider in special clauses |

## Search

Find providers by name, service type, phone, or email.

## Tips

> Use the notes field to record service ratings after each job.

> Keep service types standardized to make searching easier when scheduling maintenance.

> The **AI Assistant** can list providers by specialty when asked.

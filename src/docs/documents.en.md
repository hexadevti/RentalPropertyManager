# Documents

## What is this screen?

Stores and organizes digital files related to the portfolio — signed contracts, receipts, proof of payment, photos, insurance policies, certificates, and any other relevant documents.

## How to upload a document

Three ways to upload:

1. **Click the upload area** — opens the system file picker
2. **Drag and drop** — drag a file directly onto the indicated area
3. **Paste from clipboard** — use `Ctrl+V` after copying an image or PDF

## Form fields

- **File** — the file to upload (PDF, image, Word, Excel, etc.)
- **Name** — document title for identification in the system
- **Category** — classifies the document type:
  - Contract
  - Receipt
  - Insurance
  - Tax / Fee
  - Other
- **Link type** — who the document belongs to:
  - Property
  - Contract/Booking
  - Guest
  - Owner
- **Linked entity** — select the specific record
- **Notes** *(optional)* — annotations about the document

## Available actions

- **View** — opens the document in the browser (for PDFs and images)
- **Download** — saves the file to your computer
- **Delete** — permanently removes the file from storage

## Filters

- **Search** — by document name
- **Category** — filter by type
- **Link** — filter by related entity type

## Tips

> Always store the signed contract scan linked to the corresponding **Contract/Booking** in the system.

> PDFs are viewed directly in the browser without needing to download.

> Organize by category to facilitate future audits and searches.

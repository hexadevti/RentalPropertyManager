# Finances

## What is this screen?

Records and analyzes all financial transactions in the portfolio — rental income, maintenance expenses, taxes, insurance, and any other income or expenses related to the properties.

## Transaction types

| Type | Examples |
|---|---|
| **Income** | Rent received, forwarded condo fees, late payment penalties |
| **Expense** | Maintenance, property tax, insurance, real estate commission |

## Form fields

- **Type** — Income or Expense
- **Amount** — value
- **Category** — free-text classification (e.g., Rent, Maintenance, Tax)
- **Date** — competence date or payment date
- **Description** — free-form details
- **Property** *(optional)* — links the transaction to a specific property
- **Contract/Booking** *(optional)* — links to the corresponding contract
- **Service Provider** *(optional)* — for vendor payments

## CSV import

Use the **Import CSV** button to import transactions in bulk. Fields: type (income/expense), amount, category, description, date. Accepted date formats: `YYYY-MM-DD` or `DD/MM/YYYY`.

## Available filters

- **Month / custom period** — select the date range
- **Type** — Income, Expense, or All
- **Category** — filter by transaction type
- **Property** — transactions for a specific property
- **Contract/Booking** — transactions linked to a contract
- **Guest** — transactions associated with a tenant
- **Service Provider** — expenses with a specific vendor
- **Owner** — transactions linked to a specific owner's properties

## Financial summary

The panel displays for the selected period:
- **Total income**
- **Total expenses**
- **Net balance** (income − expenses)

Transactions are grouped by month with subtotals per period.

## Tips

> Always link the transaction to the corresponding contract for accurate property profitability reports.

> The **AI Assistant** can answer questions like "What is this month's balance?" or "What are the biggest expenses?" based on these records.

> Use standardized categories to simplify report analysis.

# Financial Transaction Form

## Type and amount

- **Type** — *Income* (money in) or *Expense* (money out)
- **Amount** — value; use period or comma as decimal separator
- **Date** — competence date or effective payment date

## Category

Classifies the transaction for report analysis. Category is free text — type anything.

Common suggestions:

| Category | Use |
|---|---|
| Rent | Monthly rent received |
| Maintenance | Repairs, renovations, upkeep |
| Tax / Fees | Property tax, municipal fees |
| Insurance | Property or tenant insurance policies |
| Condo | Condominium fees |
| Commission | Real estate or broker commission |

## Optional links

- **Property** — links the transaction to a specific property (recommended)
- **Contract/Booking** — links to the rental contract
- **Service Provider** — for vendor payments

> Fill in the links so that property profitability reports are accurate.

## Description

Free field to detail the transaction: invoice number, month reference, payment notes, etc.

## CSV import

To import multiple transactions at once, use **Import CSV** in the Finances screen. Fields: type (income/expense), amount, category, description, date.

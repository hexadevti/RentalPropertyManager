# Task Form

## Main fields

- **Title** — brief, objective description of the task (e.g., *Replace bathroom faucet*)
- **Description** — details of what needs to be done, required materials, vendor contact, etc.
- **Due date** — completion deadline; overdue tasks are highlighted in the list
- **Responsible** — select a person from the owners, guests, or service providers list

## Priority

| Level | When to use |
|---|---|
| **High** | Urgent — impacts rental or property safety |
| **Medium** | Important but not immediately urgent |
| **Low** | Routine improvements and minor adjustments |

## Status

| Status | Meaning |
|---|---|
| **Pending** | Not yet started |
| **In Progress** | Execution underway |
| **Completed** | Finished |

## Linked property

Link the task to a property so it appears in that property's reports and on the calendar.

> Tasks automatically created from inspections are pre-filled with the property and a description of the identified problem.

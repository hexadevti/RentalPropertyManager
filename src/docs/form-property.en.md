# Property Form

## Basic information

- **Name** — internal property identifier (e.g., "Apt 203 - Downtown")
- **Type** — Room, Apartment, House, or Parking
- **Capacity** — maximum number of occupants
- **Conservation state** — Excellent, Good, Fair, or Poor

## Pricing

- **Price per night** — for short-term (seasonal) contracts/bookings
- **Price per month** — for long-term (monthly) contracts

## Location

- **City** and **Address** — appear in PDF-generated contracts and on the map view

## Inspection items and furniture

- **Inspection items** — points checked at entry/exit (e.g., *Faucet*, *Window*, *Outlet*)
- **Furniture** — items present (e.g., *Queen bed*, *Wardrobe*)
- **Rooms** — property rooms (e.g., *Living room*, *Bedroom 1*, *Bathroom*) — only for apartments and houses

> Use the **Load defaults** button to automatically fill a standard list of rooms based on the property type.

## Linked owners

Select one or more owners. The link is used in owner-based reports and contract templates.

## Photos

- Drag files, paste (Ctrl+V), or click to select
- Maximum file size: 10 MB
- Set the **cover photo** by clicking the star icon — it appears on the property card
- Gallery photos display at the same size as the cover photo

## iCal Calendars

### Export link
When editing a property, the **export link** iCal is displayed — a public URL with all registered bookings. Copy and paste it into external platforms to sync availability.

### External feeds
Add iCal links from booking platforms to automatically receive reservations:

1. Select the platform (Airbnb, Booking.com, VRBO, Expedia, or Other)
2. Paste the iCal URL from the platform
3. Click **Add feed**

Feeds registered here are used by the **Sync Platforms** button in Contracts/Bookings.

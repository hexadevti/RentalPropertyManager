# Formulário de Documento

## Upload do arquivo

Três formas de enviar:

1. **Clique na área** — abre o seletor de arquivos do sistema
2. **Arraste e solte** — arraste diretamente para a área de upload
3. **Cole** com `Ctrl+V` — após copiar uma imagem ou PDF da área de transferência

Formatos aceitos: PDF, imagens (JPG, PNG), Word, Excel e outros.

## Identificação

- **Nome** — título do documento exibido na lista (ex: *Contrato assinado - João Silva - Jan/2025*)
- **Categoria** — classifica o tipo de arquivo:

| Categoria | Exemplos |
|---|---|
| Contrato | Contrato de locação, aditivo, rescisão |
| Recibo | Comprovante de pagamento, recibo de aluguel |
| Seguro | Apólice de seguro do imóvel ou do inquilino |
| Imposto / Taxa | IPTU, condomínio, certidões |
| Outro | Fotos, laudos, declarações diversas |

## Vínculo do documento

Define a quem pertence o documento:

- **Propriedade** → selecione o imóvel
- **Contrato** → selecione o contrato de locação
- **Hóspede** → selecione o inquilino
- **Proprietário** → selecione o dono do imóvel

> Vincule sempre ao registro mais específico — ex: um contrato assinado deve ser vinculado ao **Contrato**, não apenas à propriedade.

## Observações

Campo livre para anotações: número de versão, data de validade, quem assinou etc.

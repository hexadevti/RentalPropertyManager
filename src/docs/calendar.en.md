# Calendar

## What is this screen?

Unified view of all system events in monthly calendar format — active contracts, payment due dates, appointments, and tasks.

## Navigation

- Use **◀ ▶** arrows to navigate between months
- Click any **day** to see event details for that day

## Displayed event types

| Color / Icon | Type |
|---|---|
| **Contracts/Bookings** | Active rental periods (start and end) |
| **Payments** | Rent due dates |
| **Appointments** | Appointments registered in the Schedule module |
| **Tasks** | Tasks with a due date on that day |

## Side panel

The calendar displays laterally:
- **Active contracts/bookings** — list of ongoing rentals with property and guest
- **Upcoming appointments** — future appointments in chronological order

## Quick actions

- **New contract/booking** — create a contract directly from the calendar
- **New appointment** — schedule a visit or maintenance from the calendar

## Property mini-calendars

For portfolios with multiple properties, the screen displays individual mini-calendars per property, making it easy to see availability for each unit.

## Tips

> Use the calendar to check period conflicts before creating new short-term contracts.

> Payment due dates appear on the day configured in the contract as the **payment due day**.

> Click an event to see full details and navigate to the corresponding record.

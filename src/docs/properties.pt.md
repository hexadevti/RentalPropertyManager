# Propriedades

## O que é esta tela?

Gerenciaa o portfólio de imóveis disponíveis para locação. Cada propriedade é a unidade base do sistema — contratos/reservas, vistorias, finanças e tarefas são sempre vinculados a uma propriedade.

## Tipos de propriedade

| Tipo | Descrição |
|---|---|
| **Quarto** | Unidade individual dentro de uma casa ou apartamento compartilhado |
| **Apartamento** | Unidade autônoma em edifício |
| **Casa** | Imóvel residencial completo |
| **Estacionamento** | Vaga ou garagem avulsa |

## Cadastrando uma propriedade

### Informações básicas
- **Nome** — identificação interna do imóvel (ex: "Apto 203 - Centro")
- **Tipo** — define a estrutura do checklist de vistoria
- **Capacidade** — número máximo de ocupantes
- **Estado de conservação** — Excelente, Bom, Regular ou Ruim

### Preços
- **Preço por noite** — para locação temporária (temporada)
- **Preço mensal** — para locação de longo prazo

### Localização
- **Cidade** e **Endereço** — usados em contratos/reservas, relatórios e mapa

### Itens de vistoria e mobiliário
- **Itens de vistoria** — pontos a verificar na entrada/saída (ex: "Torneira", "Vidro", "Parede")
- **Mobiliário** — itens presentes no imóvel (ex: "Cama", "Armário", "Geladeira")
- **Ambientes** — cômodos da propriedade (ex: "Sala", "Quarto 1", "Banheiro") — apenas para apartamentos e casas

> Os itens de vistoria e ambientes definem o checklist automático nas vistorias digitais.

### Proprietários vinculados
Associe um ou mais proprietários ao imóvel para controle de repasse e relatórios por proprietário.

### Fotos
Carregue fotos do imóvel (até 10 MB por arquivo). Defina a **foto de capa** que aparece no card da propriedade. Arraste para reordenar. Suporta colar imagens (Ctrl+V) e arrastar arquivos.

## Calendários iCal

A seção **Calendários iCal** na propriedade permite dois tipos de integração:

### Link de exportação (nosso sistema → plataformas)
Cada propriedade gera um link iCal público com todas as reservas cadastradas. Copie o link e cole no Airbnb, Booking.com ou qualquer app de calendário para sincronizar disponibilidade.

### Feeds externos (plataformas → nosso sistema)
Adicione links iCal de plataformas externas (Airbnb, Booking.com, VRBO, Expedia):
1. Clique em **Adicionar feed**
2. Selecione a plataforma
3. Cole a URL iCal fornecida pela plataforma
4. Salve a propriedade

Os feeds cadastrados são usados pelo botão **Sincronizar Plataformas** na tela de Contratos/Reservas.

## Importação via CSV

Use o botão **Importar CSV** para importar múltiplas propriedades de uma vez. Baixe o template, preencha e importe.

## Visualização em mapa

O botão **Mapa** exibe as propriedades georreferenciadas (requer endereço cadastrado). Clique em **Mostrar no mapa** no card para focar uma propriedade específica.

## Ações disponíveis

- **Editar** — atualiza os dados da propriedade
- **Gerar Contrato/Reserva** — abre o formulário pré-preenchido com esta propriedade
- **Copiar link iCal** — copia o link de exportação da propriedade
- **Duplicar** — cria uma cópia da propriedade com os mesmos dados
- **Excluir** — remove a propriedade

## Dicas

> Preencha os ambientes corretamente — eles definem a estrutura do checklist de vistoria.

> Para sync com Airbnb/Booking.com, cadastre o feed iCal da plataforma e use o botão **Sincronizar Plataformas** em Contratos/Reservas.

> Você pode vincular a mesma propriedade a múltiplos proprietários quando há copropriedade.

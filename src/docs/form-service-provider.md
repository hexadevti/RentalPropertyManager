# Formulário de Prestador de Serviço

## Identificação

- **Nome** — nome completo (autônomo) ou razão social (empresa)
- **Tipo de serviço** — especialidade principal (ex: *Elétrica*, *Hidráulica*, *Limpeza*, *Pintura*, *Geral*)

> Use nomes de serviço padronizados para facilitar a busca ao agendar manutenções.

## Contato

- **Telefone** — com DDD; número principal de contato
- **E-mail** — para envio de solicitações e orçamentos

## Documentação fiscal

- **CPF / CNPJ** — necessário para emissão de notas fiscais e contratos de serviço

## Endereço

Endereço do prestador ou empresa — para correspondências e contratos formais.

## Observações

Use este campo para:
- Nota de avaliação após cada serviço (ex: *⭐⭐⭐⭐ — bom atendimento*)
- Preços médios praticados
- Restrições de horário ou acesso
- Contato de um responsável específico

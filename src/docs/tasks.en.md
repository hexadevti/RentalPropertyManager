# Tasks

## What is this screen?

Manages activities, pending items, and maintenance related to the portfolio. Each task can be prioritized, assigned to a responsible party, and tracked by execution status.

## Form fields

- **Title** — brief description of the task
- **Description** — details of what needs to be done, conditions, required materials, etc.
- **Due date** — completion deadline
- **Priority** — High, Medium, or Low
- **Status** — Pending, In Progress, or Completed
- **Responsible** — name of the person who will execute the task
- **Property** *(optional)* — property related to the task

## Priority levels

| Priority | Suggested use |
|---|---|
| **High** | Urgent — impacts rental or property safety |
| **Medium** | Important but not immediately urgent |
| **Low** | Routine improvements and minor adjustments |

## Lifecycle status

```
Pending → In Progress → Completed
```

## Overdue tasks

Tasks with a past due date and a status other than "Completed" are visually marked as **overdue** to facilitate prioritization.

## Automatic creation from inspections

In **Inspections**, when a negative difference is identified on an item (condition worsened), the **Create task** button automatically generates a task linked to the property with the details of the problem found.

## Search and filters

- **Search** — by title or description
- **Property** — filter by property
- **Status** — Pending, In Progress, Completed, or All

## Tips

> Use **High** priority tasks for issues that prevent rental (leak, broken equipment, etc.).

> Always link the property to facilitate visualization in the **Calendar** and **Reports**.

> The **AI Assistant** can list pending tasks by property when asked.

# Owner Form

## Identification

- **Name** — full name (individual) or company name (legal entity)
- **Email** — primary contact for sending reports and communications
- **Phone** — with area code

## Documentation

Add multiple documents (CPF, CNPJ, ID, passport, etc.), each with type and number.

> Click **Add document** to include more than one document.

## Personal data

- **Nationality** — Brazilian or foreign
- **Marital status** — single, married, divorced, or widowed
- **Profession** — current occupation

## Address

Full address for correspondence and issuance of formal documents.

## Notes

Internal notes: transfer agreements, co-ownership percentages, contact preferences, etc.

> To link the owner to a property, edit the **Property** and add this owner in the "Linked owners" section.

## CSV import

To register multiple owners at once, use **Import CSV** in the Owners screen.

# Calendário

## O que é esta tela?

Visão unificada de todos os eventos do sistema em formato de calendário mensal — contratos ativos, vencimentos de pagamento, agendamentos e tarefas.

## Navegação

- Use as setas **◀ ▶** para navegar entre meses
- Clique em qualquer **dia** para ver os detalhes dos eventos daquele dia

## Tipos de eventos exibidos

| Cor / Ícone | Tipo |
|---|---|
| **Contratos** | Períodos de locação ativa (início e fim) |
| **Pagamentos** | Dias de vencimento de aluguel |
| **Agendamentos** | Compromissos cadastrados na Agenda |
| **Tarefas** | Tarefas com prazo no dia |

## Painel lateral

O calendário exibe lateralmente:

- **Contratos ativos** — lista de locações em andamento com imóvel e hóspede
- **Próximos compromissos** — agendamentos futuros em ordem cronológica

## Ações rápidas

- **Novo contrato** — cria um contrato diretamente pelo calendário
- **Novo compromisso** — agenda uma visita ou manutenção a partir do calendário

## Mini-calendários por propriedade

Para portfólios com múltiplos imóveis, a tela exibe mini-calendários individuais por propriedade, facilitando a visualização de disponibilidade de cada unidade.

## Dicas

> Use o calendário para verificar conflitos de período antes de criar novos contratos de temporada.

> Os vencimentos de pagamento aparecem no dia configurado no contrato como **dia de vencimento**.

> Clique no evento para ver os detalhes completos e navegar para o registro correspondente.

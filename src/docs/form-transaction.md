# Formulário de Transação Financeira

## Tipo e valor

- **Tipo** — *Receita* (entrada de dinheiro) ou *Despesa* (saída de dinheiro)
- **Valor** — montante em reais; use ponto ou vírgula como separador decimal
- **Data** — data de competência ou data do pagamento efetivo

## Categoria

Classifica a transação para análise nos relatórios:

| Categoria | Uso |
|---|---|
| Aluguel | Recebimento de aluguel mensal |
| Manutenção | Reparos, reformas, conservação |
| IPTU / Taxas | Impostos e taxas municipais |
| Seguro | Apólices de seguro do imóvel |
| Condomínio | Taxas condominiais |
| Comissão | Comissão de imobiliária ou corretor |
| Outros | Qualquer outra movimentação |

## Vínculos opcionais

- **Propriedade** — vincula a transação a um imóvel específico (recomendado)
- **Contrato** — vincula ao contrato de locação correspondente
- **Prestador de serviço** — para despesas com fornecedores

> Preencha os vínculos para que os relatórios de rentabilidade por propriedade sejam precisos.

## Descrição

Campo livre para detalhar a transação: número de nota fiscal, referência do mês, observações do pagamento etc.

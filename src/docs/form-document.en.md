# Document Form

## File upload

Three ways to upload:

1. **Click the area** — opens the system file picker
2. **Drag and drop** — drag directly onto the upload area
3. **Paste** with `Ctrl+V` — after copying an image or PDF to the clipboard

Accepted formats: PDF, images (JPG, PNG), Word, Excel, and others.

## Identification

- **Name** — document title shown in the list (e.g., *Signed Contract - John Smith - Jan/2025*)
- **Category** — classifies the file type:

| Category | Examples |
|---|---|
| Contract | Rental contract, addendum, termination |
| Receipt | Payment proof, rent receipt |
| Insurance | Property or tenant insurance policy |
| Tax / Fee | Property tax, condo fees, certificates |
| Other | Photos, reports, declarations |

## Document link

Defines who the document belongs to:

- **Property** → select the property
- **Contract/Booking** → select the rental contract
- **Guest** → select the tenant
- **Owner** → select the property owner

> Always link to the most specific record — e.g., a signed contract should be linked to the **Contract/Booking**, not just the property.

## Notes

Free field for annotations: version number, expiration date, who signed, etc.
